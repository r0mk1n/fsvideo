# fsvideo
Script for proportional resize **youtube** and **vimeo** video iframes to size of parent element

###this script required jquery###

## How to use
include script:

````<script src="./fsvideo.js"></script>````

and initialize:
```
<script>
$(document).ready(function() {
	var fsvideo = new FSVideo();
	fsvideo.watch();
})
</script>	
````
